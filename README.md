# 安装及启动
## 1. 安装依赖 
`npm install`
## 2. 启动项目(请保证nodejs的版本在7.0以上)
`npm start`
## 3. 运行网页
### 浏览器输入 `localhost:8080`
## 3. 打包项目
`npm run build`
# 介绍
## 1. 功能介绍
#### 实现组件懒加载功能，提取第三方库，样式的autoprefixer，样式采用less以及css modules
#### 通过glob实现支持多入口配置，详见webpack添加
#### 为后台管理系统量身打造的框架
